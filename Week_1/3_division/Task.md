# Деление

Дано два целых числа A и B в диапазоне от 0 до 1 000 000 включительно. Напишите программу, которая вычисляет целую часть частного от деления A на B.

Если B = 0, выведите "Impossible".

## Пример

| stdin |   stdout   |
| :---: | :--------: |
| 10 2  |     5      |
|  3 5  |     0      |
| 11 0  | Impossible |
